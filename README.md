# Datenschutzerklärung – Sleep Coach

## 1. Verantwortlicher
Sleep Coach  
E-Mail: contact@sleepcoach.app

## 2. Allgemeine Hinweise
Der Schutz deiner persönlichen Daten ist uns wichtig. Diese App verarbeitet Daten ausschließlich im Rahmen der gesetzlichen Bestimmungen (DSGVO) und nur in dem Umfang, der für ihren Betrieb notwendig ist.

## 3. Welche Daten werden verarbeitet?

### a) Lokale Speicherung
Beim Verwenden der App werden lokal auf deinem Gerät gespeichert:
- Schlaf-Feedback (z. B. schlecht / okay / gut)
- Einstellungen innerhalb der App

Diese Daten bleiben ausschließlich auf deinem Gerät und werden nicht an uns übertragen.

### b) Keine Registrierung
Die App erfordert keine Anmeldung. Es werden keine Namen, E-Mail-Adressen oder Passwörter erfasst.

### c) Keine medizinischen Daten
Sleep Coach ist kein medizinisches Produkt. Die eingegebenen Angaben dienen ausschließlich der Erstellung von Entspannungsroutinen.

## 4. Server
Derzeit erfolgt keine Speicherung personenbezogener Daten auf Servern.

## 5. YouTube
In der App befindet sich ein Link zu einem YouTube-Kanal. Beim Öffnen gelten die Datenschutzbestimmungen von YouTube / Google.

## 6. Premium & Zahlungen
Zahlungen werden, sobald aktiviert, über den Apple App Store und Google Play abgewickelt. Zahlungsdaten werden nur von diesen Plattformen verarbeitet.

## 7. Analyse & Tracking
Es werden keine Analyse- oder Tracking-Tools verwendet.

## 8. Rechte
Du hast jederzeit das Recht auf Auskunft, Löschung oder Berichtigung deiner Daten.
